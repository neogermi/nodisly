# Install Nodisly

## (1) Install Redis as database

http://redis.io/download

## (2) Install NodeJS

http://nodejs.org/

### (2.1) Install NodeJS dependencies

```
$ npm install -g redis
$ npm install -g express
$ npm install -g nodemailer
```
# Run Nodisly

## (1) Set up database

```
$ node fill_keys.js
```

## (2) Run nodisly

```
$ node nodisly.js
```
